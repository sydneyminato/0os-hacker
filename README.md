# 0os-hacker

**"Nothing sounds until it moves."**

## Overview
このアプリケーションは、Web技術（HTML/JS）とRust（Tauri）を融合させた「Edited Reality」への介入ツールです。
このリポジトリには音声ファイルは含まれていません。閲覧するだけでは無音であり、あなたの手によって「音」が与えられるのを待っています。

## How to use
1. アプリケーションを起動します。
2. **"BEAST"** ボタンを押し、ベースとなる咆哮や環境音（WAV/MP3）をロードします。
3. **"MAESTRO"** ボタンを押し、重なり合うオーケストラや旋律（WAV/MP3）をロードします。
4. 中央のスライダーを動かしてください。あなたの介入に合わせて2つの世界が混ざり合い、0OSの響きが生まれ始めます。

## Philosophy
0OS（Zero-OS）において、沈黙は欠陥ではありません。
あなたが「素材」を供給し、自ら動かさない限り、世界は存在しないも同然だからです。
このツールは、あなたの現実を編集するための最初のデバイスとなります。
